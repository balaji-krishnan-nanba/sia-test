## Critical Fixes Applied

### 1. Answer Pattern Randomization (December 7, 2025)

**Problem:** Many files had exploitable answer patterns:
- Perfect A→B→C→D rotation (100% predictable)
- Sequential runs (A,A,A,A or B,B,B,B)

**Files Fixed:** 25+ files across all 4 exam types

| Exam | Files Fixed | Original Issue |
|------|-------------|----------------|
| CP Units 1-4 | 8 files | Perfect ABCD rotation |
| DS Units 3-4 | 11 files | Sequential runs |
| SG | 7 files | Perfect rotation |
| CCTV | 4 files | Perfect rotation |

**Example Fix:**
```
Before: A,B,C,D,A,B,C,D,A,B,C,D... (100% exploitable)
After:  B,D,C,B,A,B,D,A,C,D,A,B... (random)
```

### 2. PACE Section 24A Clarification (DS Ch 2.4 Q3)

**Problem:** Original explanation was unclear about the TWO separate requirements.

**Fix Applied:**
```
NEW EXPLANATION:
Section 24A PACE has TWO separate requirements for a lawful citizen's arrest
of a completed offence:
(1) The OFFENCE must have actually been committed - this is a matter of FACT,
    not suspicion
(2) You must have reasonable grounds to SUSPECT the person is guilty - you
    don't need to be certain, but your suspicion must be reasonable

KEY DISTINCTION: If no offence actually occurred, the arrest is unlawful
regardless of how reasonable your belief was. However, if an offence did
occur and you arrest the wrong person, the arrest may still be lawful if
your suspicion of that person was reasonable at the time.
```

### 3. Tactical Breathing Correction (CP Ch 7.3)

**Problem:** Only 3 phases described (should be 4)

**Fix Applied:**
```
Tactical breathing uses a 4-phase cycle:
1. Breathe IN for 4 seconds
2. HOLD for 4 seconds
3. Breathe OUT for 4 seconds
4. HOLD for 4 seconds
Then repeat (4-4-4-4 pattern, also called "box breathing")
```

### 4. Mehrabian Research Context (CP Ch 7.3)

**Problem:** Percentages stated without context about their limited applicability.

**Fix Applied:**
Added note: *"This research specifically examined the communication of feelings
and attitudes in ambiguous contexts, not all forms of communication—these
percentages should not be applied universally."*

### 5. "If They Can Talk, They Can Breathe" Myth (CP Ch 7.5)

**Problem:** Dangerous myth not explicitly refuted.

**Fix Applied:**
```
The myth that 'if they can talk, they can breathe' is FALSE and dangerous—
people can speak with severely restricted breathing and can die even while
speaking or shortly after speaking.
```

### 6. GHB Reclassification (DS Ch 2.5, SG Ch 2.7)

**Problem:** GHB listed as Class C (outdated).

**Fix Applied:**
```
GHB was reclassified from Class C to Class B on 10 April 2023 due to its
role in drug-facilitated sexual assault and potential for serious harm.
```

### 7. Surveillance Camera Commissioner Update (CCTV Ch 2.2)

**Problem:** Referenced old title.

**Fix Applied:**
```
The Biometrics and Surveillance Camera Commissioner oversees the Surveillance
Camera Code of Practice. This role replaced the former Surveillance Camera
Commissioner in 2022.
```

### 8. Four D's Disambiguation (CP Ch 4.3, 4.4)

**Problem:** Two different "4 D's" frameworks confused.

**Fix Applied:**
- **Four D's of Reporting:** Discover, Describe, Document, Deliver
- **4 D's of Protective Security:** Deter, Detect, Delay, Deny

## Answer Pattern Guidelines

### Acceptable Patterns
- Random distribution across chapter
- No more than 2 consecutive same answers
- Each letter (A,B,C,D) appears 5-6 times per 20 questions
- No perfect rotation (A,B,C,D,A,B,C,D...)

### Exploitable Patterns to Avoid
```
❌ A,B,C,D,A,B,C,D,A,B,C,D... (perfect rotation)
❌ A,A,A,A,B,B,B,B,C,C,C,C... (sequential blocks)
❌ A,B,A,B,A,B,A,B... (alternating pairs)
❌ D,D,D,D,D... (5+ same in a row)
```

### Checking Pattern Exploitability
```python
def check_exploitability(answers):
    # Check rotation match
    rotation_match = sum(1 for i, a in enumerate(answers) if a == 'ABCD'[i % 4])
    rotation_pct = rotation_match / len(answers)

    # Check max run length
    max_run = max(len(list(g)) for _, g in groupby(answers))

    # Score: 0 = good, 100 = fully exploitable
    return (rotation_pct * 50) + (min(max_run * 15, 50))
```

## Factual Accuracy Notes

### Key Legal References

| Topic | Correct Reference | Notes |
|-------|-------------------|-------|
| SIA Creation | Private Security Industry Act 2001 | Not 2003 or other years |
| Citizen's Arrest | PACE 1984 Section 24A | "Any person" arrest power |
| Privacy Right | Human Rights Act Article 8 | Private & family life |
| Torture Prohibition | Human Rights Act Article 3 | Absolute right |
| CCTV Code | Protection of Freedoms Act 2012 | Surveillance Camera Code |
| Data Protection | UK GDPR + Data Protection Act 2018 | Post-Brexit framework |

### Key Factual Points

| Topic | Correct Information |
|-------|---------------------|
| SIA Licence Duration | 3 years |
| SIA Licence Fee | £184 (as of 2024) |
| GHB Classification | Class B (since 10 April 2023) |
| Mehrabian 55-38-7 | Only applies to emotional/attitude communication |
| Positional Asphyxia | Prone position highest risk; "can talk = can breathe" is FALSE |
| Tactical Breathing | 4-4-4-4 (in-hold-out-hold) |

### Commonly Confused Terms

| Term 1 | Term 2 | Difference |
|--------|--------|------------|
| Four D's of Reporting | 4 D's of Protective Security | Discover/Describe/Document/Deliver vs Deter/Detect/Delay/Deny |
| Reactive Aggression | Instrumental Aggression | Emotional/impulsive vs Calculated/goal-directed |
| Cover Block | Deflection/Parry | Survival posture vs Redirecting attack |
