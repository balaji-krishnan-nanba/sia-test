TOPIC 2.8.4: RADIO COMMUNICATIONS
--------------------------------------------------------------------------------

Radio Equipment:
- Personal radios (PMR)
- Earpieces (covert)
- Lapel microphones
- Encrypted systems
- Backup equipment

Radio Discipline:

CLARITY:
- Speak clearly
- Normal pace
- Moderate volume
- Face away from noise

BREVITY:
- Keep messages short
- Essential info only
- No casual chat
- Quick exchanges

SECURITY:
- Assume monitored
- No sensitive info in clear
- Use code words
- Principal's name not used

PROCEDURE:
- Follow protocols
- Standard phrases
- Proper call signs
- Acknowledge receipt

Radio Procedure:

INITIATING CALL:
"[Receiver call sign], this is [Sender call sign], over"
Example: "Alpha 1, this is Alpha 2, over"

ACKNOWLEDGING:
"[Sender call sign], this is [Receiver call sign], send, over"
Example: "Alpha 2, this is Alpha 1, send, over"

SENDING MESSAGE:
"[Message], over"
Example: "Package moving to vehicle, over"

CONFIRMING RECEIPT:
"Roger" (received and understood)
"Say again" (repeat message)
"Wilco" (will comply)

ENDING TRANSMISSION:
"Out"

Common Radio Terms:

ROGER:
- Received and understood

SAY AGAIN:
- Repeat last message

WAIT:
- Pause, will respond

WAIT OUT:
- Longer pause

SEND:
- Go ahead with message

WILCO:
- Will comply (do as instructed)

NEGATIVE:
- No

AFFIRMATIVE:
- Yes

OUT:
- End of transmission

Radio Checks:
- Pre-operation
- Regular intervals
- After issues
- All team members

Radio Problems:
- Dead areas
- Interference
- Battery failure
- Frequency congestion
- Backup procedures

TOPIC 2.8.5: CODE WORDS AND CALL SIGNS
--------------------------------------------------------------------------------

Why Code Words:
- Security
- Brevity
- Clarity
- Conceal meaning from others

Types of Code Words:

CALL SIGNS:
- Names for team members
- Alpha 1, Alpha 2, etc.
- Positions (Point, Rear)
- Changed regularly

LOCATION CODES:
- Venues
- Routes
- Rally points
- Sensitive locations

STATUS CODES:
- Principal position
- Team status
- Alert levels
- Activity status

ACTION CODES:
- Commands
- Warnings
- Status changes
- Emergency triggers

Common Code Concepts:

PRINCIPAL CODE:
- "Package"
- "Principal"
- "VIP"
- Never real name on radio

LOCATION CODE:
- "Base" = home
- "Office" = workplace
- Numbers for venues
- Pre-agreed meanings

STATUS CODE:
- "Green" = all clear
- "Amber" = caution
- "Red" = emergency
- "Secure" = area safe

ACTION CODE:
- "Move" = depart
- "Hold" = wait
- "Abort" = cancel
- "Rally" = regroup

Code Word Principles:
- Pre-briefed
- Known to all team
- Not obvious meaning
- Changed if compromised
- Documented securely

Example Communication:
"Alpha 1, Alpha 2, Package moving to vehicle, over"
(TL, this is CPO, Principal going to car)

"Alpha 2, Alpha 1, roger, out"
(CPO, TL, understood, end)

TOPIC 2.8.6: HAND SIGNALS
--------------------------------------------------------------------------------

Why Hand Signals:
- Silent communication
- Noise environments
- Radio failure backup
- Covert operations
- Quick visual cues

Common Hand Signals:

STOP:
- Open palm raised
- Facing recipient
- Arm extended
- Immediate stop

GO/MOVE:
- Wave forward
- Palm forward
- Beckoning motion
- Proceed

COME TO ME:
- Beckoning gesture
- Palm up
- Fingers curl
- Move to signaller

GET DOWN:
- Palm down
- Pressing motion
- Downward movement
- Take cover/crouch

THREAT/ENEMY:
- Point at threat
- Or closed fist
- Direction indicated
- Alert team

OKAY/ALL CLEAR:
- Thumbs up
- Or OK sign
- Positive confirmation
- Status good

NUMBERS:
- Fingers for 1-5
- Both hands for more
- Direction and count
- Identify threats

Directional Signals:
- Point in direction
- Clear indication
- Accompanied by other signals
- Team looks where indicated

Signal Principles:
- Pre-agreed meanings
- Visible to team
- Discreet from others
- Backup for voice
- Regular practice

When to Use:
- Cannot use radio
- Noise too loud
- Need silence
- Quick communication
- Supplement verbal

Limitations:
- Line of sight needed
- Can be misunderstood
- Limited vocabulary
- Not always appropriate
- Training required

Question 1 [Easy]
What does "ROGER" mean in radio communications?

A) Send your message
B) Wait for further instructions
C) Received and understood
D) End of transmission

Correct Answer: C

Explanation: "Roger" is the standard radio term meaning "received and 
understood." It confirms that the message has been heard and comprehended. 
Other key terms include "Wilco" (will comply), "Say again" (repeat), and 
"Out" (end of transmission).

- Follow unless unsafe
- Initiative within role

COMMON DISTRACTORS:
