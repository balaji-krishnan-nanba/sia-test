TOPIC 4.5.1: IMMEDIATE RESPONSE PRINCIPLES
--------------------------------------------------------------------------------

Priority Order in ANY Incident:

1. PRINCIPAL SAFETY:
   - Always first priority
   - Every action aimed at this
   - Nothing more important

2. TEAM SAFETY:
   - CPO must survive to protect
   - Support team members
   - Coordinate response

3. OTHERS:
   - Public safety
   - Duty of care
   - Where possible

4. PROPERTY:
   - Lowest priority
   - Replace property, not lives
   - Only if safe

Key Response Principles:

ACT IMMEDIATELY:
- No hesitation
- Trained response
- Split-second decisions
- Action saves lives

MOVE TO SAFETY:
- Away from threat
- Cover and concealment
- Pre-identified locations
- Continuous movement

COMMUNICATE:
- Team awareness
- Emergency services
- Principal informed
- Clear messages

Decision Making Under Pressure:

OODA LOOP:
- Observe (what's happening)
- Orient (understand situation)
- Decide (choose action)
- Act (execute decision)

TRAINING HELPS:
- Pre-programmed responses
- Automatic actions
- Reduces thinking time
- Confidence

Don't Freeze:
- Action better than inaction
- Move, don't stay static
- Trained response
- Trust training

TOPIC 4.5.2: RUN-HIDE-TELL
--------------------------------------------------------------------------------

What is Run-Hide-Tell:
- UK guidance for terror attacks
- Applies to firearms and weapons attacks
- Sequential approach
- Save lives guidance

RUN:

FIRST PRIORITY:
- If safe to do so
- Best chance of survival
- Distance = safety

HOW TO RUN:
- Away from threat
- Fastest safe route
- Don't stop for belongings
- Insist others come
- Don't let others slow you

CPO APPLICATION:
- Move principal immediately
- Take decisive action
- Don't wait for agreement
- Physical guidance if needed

HIDE:

WHEN TO HIDE:
- Cannot run safely
- Threat between you and exit
- Better option than running

WHERE TO HIDE:
- Find COVER (stops bullets)
- Not just concealment
- Solid barriers
- Locked/barricaded room

HOW TO HIDE:
- Lock/barricade door
- Stay quiet
- Silence phone (not just vibrate)
- Stay hidden
- Don't come out until certain

COVER vs CONCEALMENT:
- Cover = physical protection
- Concealment = visual only
- Seek COVER in weapons attack

TELL:

WHEN SAFE:
- Only when safe to do so
- Don't risk yourself
- Information saves lives

CALL 999:
- Location
- What's happening
- Number of attackers
- Description
- Weapons seen
- Direction of travel
- Victims if known

STAY ON LINE:
- Until told to hang up
- Answer questions
- Update information

For CPOs During Run-Hide-Tell:

RUN PHASE:
- Principal moves WITH you
- Physical control if needed
- Team coordinates
- Pre-planned routes

HIDE PHASE:
- Secure location
- Principal protected
- Communication maintained
- Ready to run again

TELL PHASE:
- Report situation
- Request response
- Give updates
- Coordinate with team

TOPIC 4.5.3: RESPONSE TO SPECIFIC ATTACK TYPES
--------------------------------------------------------------------------------

Firearms Attack:

IMMEDIATE RESPONSE:
- Run-Hide-Tell
- Get principal to cover
- Distance from shooter
- Evacuate if possible

COVER OPTIONS:
- Concrete walls
- Engine blocks
- Thick brick
- Earth banks
- NOT: Glass, drywall, cars (doors), furniture

IF SHOOTER NEARBY:
- Hide immediately
- Barricade if possible
- Silence and stillness
- Last resort: Fight

NEVER:
- Run towards gunfire
- Try to reason with shooter
- Assume it's over

Explosion/IED:

IMMEDIATE RESPONSE:
- Move away from blast area
- Expect secondary devices
- Cover from debris
- Check for injuries

SECONDARY DEVICES:
- Common tactic
- Target responders
- Different location
- Stay alert

EVACUATION:
- Away from blast site
- Not through area
- Watch for hazards
- Assembly point

Vehicle Attack:

IMMEDIATE RESPONSE:
- Get off road/pavement
- Move perpendicular to vehicle
- Seek solid cover
- Don't run in vehicle's path

COVER:
- Behind solid structures
- Substantial bollards
- Inside buildings
- Elevated positions

AFTER INITIAL ATTACK:
- Attacker may exit vehicle
- Secondary attack possible
- Continue vigilance
- Evacuate area

Edged Weapon Attack:

IMMEDIATE RESPONSE:
- Distance is key
- Run if possible
- Barriers between threat
- Don't engage unless necessary

BARRIERS:
- Doors
- Vehicles
- Furniture
- Anything solid

LAST RESORT:
- Fight back
- Improvised weapons
- Commit fully
- Aim for vulnerable areas

--------------------------------------------------------------------------------

When to Evacuate:

IMMEDIATELY:
- Threat in area
- Attack in progress
- Fire/explosion
- Suspect device nearby

CONSIDER:
- Route safety
- Alternative exits
- Threat location
- Secondary risks

Evacuation Principles:

SPEED:
- Move quickly
- Don't delay
- Principal safety first

ROUTE:
- Away from threat
- Clear path
- Pre-identified if possible
- Avoid crowds/bottlenecks

DESTINATION:
- Safe location
- Pre-planned if possible
- Police RVP
- Away from danger

Types of Evacuation:

PLANNED:
- Orderly exit
- Pre-identified routes
- Assembly points
- Controlled movement

EMERGENCY:
- Immediate exit
- Life threat
- Speed priority
- May be chaotic

FIGHTING WITHDRAWAL:
- Under attack
- Maintain cover
- Team supports
- Continuous movement

Evacuation Route Selection:

CONSIDER:
- Direction of threat
- Clear pathways
- Distance to safety
- Secondary risks

AVOID:
- Toward threat
- Bottlenecks
- Predictable routes
- Glass frontages

Principal Evacuation:

CPO ROLE:
- Lead principal
- Physical guidance
- Cover from threat
- Communication

PRINCIPAL COMPLIANCE:
- May be shocked
- Clear instructions
- Physical control
- Keep moving

TEAM COORDINATION:
- Cover and move
- Communication
- Support each other
- Rally points

TOPIC 4.5.5: COVER AND CONCEALMENT
--------------------------------------------------------------------------------

Understanding the Difference:

COVER:
- Physical protection
- Stops bullets/attacks
- Substantial barrier
- Primary choice

CONCEALMENT:
- Hides from view
- Does NOT stop bullets
- Visual barrier only
- Better than nothing

Types of Cover:

EFFECTIVE COVER:
- Concrete walls
- Engine blocks of vehicles
- Thick brick
- Earth banks
- Steel structures

PARTIAL COVER:
- Trees (larger)
- Furniture (heavy)
- Car wheels
- Corners of buildings

NOT COVER:
- Car doors
- Glass
- Drywall/plasterboard
- Thin walls
- Bushes

Concealment Examples:
- Curtains
- Vegetation
- Darkness
- Thin walls
- Furniture

Using Cover:

PRINCIPLES:
- Get low
- Get behind
- Stay behind
- Move if necessary

MOVEMENT:
- Cover to cover
- Quick dashes
- Team coordination
- Don't expose unnecessarily

ANGLES:
- Distance from cover = better angle protection
- Too close exposes from sides
- Position matters

Body Position:
- Make yourself small
- Present minimum target
- Behind cover fully
- Head and limbs protected

TOPIC 4.5.6: COORDINATION WITH EMERGENCY SERVICES
--------------------------------------------------------------------------------

During Incident:

CALLING 999:
- Clear location
- What's happening
- Threat description
- Casualties
- Your location

INFORMATION:
- Number of attackers
- Weapons seen
- Direction of movement
- Areas to avoid
- Principal status

STAY ON LINE:
- Updates requested
- Instructions given
- Location changes
- Threat changes

When Police Arrive:

ARMED RESPONSE:
- Follow ALL instructions
- Hands visible
- No sudden movements
- Drop anything in hands
- Get on ground if told

IDENTIFICATION:
- May be treated as threat initially
- Comply first, explain later
- Show ID when safe
- Be patient

DON'T:
- Run towards police
- Argue
- Reach for anything
- Make sudden movements
- Point at officers

Communication:
- Clear and calm
- Answer questions
- Give information
- Report principal status
- Coordinate handover

Handover to Police:

WHEN:
- Area secure
- Police have control
- Principal safe
- Appropriate time

INFORMATION:
- Full briefing
- What happened
- What you saw
- Actions taken
- Ongoing concerns

STATEMENT:
- May be required
- Factual account
- Contemporaneous notes
- Professional

KEY POINTS FOR THIS CHAPTER
================================================================================

MUST GET CORRECT:

PRIORITY ORDER IN INCIDENTS:
1. Principal safety (ALWAYS FIRST)
2. Team safety
3. Others
4. Property (LOWEST)

RUN-HIDE-TELL:
- RUN = Evacuate immediately if safe
- HIDE = If cannot run, find COVER (not just concealment)
- TELL = When safe, call 999 with details

COVER vs CONCEALMENT:
- Cover = STOPS bullets (concrete, engine block, thick brick)
- Concealment = HIDES from view only (does NOT stop bullets)

EFFECTIVE COVER EXAMPLES:
- Concrete walls
- Engine blocks
- Thick brick
- Earth banks

NOT COVER:
- Car doors
- Glass
- Drywall/plasterboard
- Thin walls
- Bushes

WHEN POLICE ARRIVE:
- Follow ALL instructions
- Hands visible
- No sudden movements
- Drop anything in hands
- Comply first, explain later

RESPONSE TO VEHICLE ATTACK:
- Move PERPENDICULAR to vehicle path
- Don't run in same direction as vehicle
- Seek solid cover

SECONDARY DEVICES:
- Common terrorist tactic
- Target first responders
- Stay alert after initial attack
- Different location possible

OODA LOOP:
- Observe, Orient, Decide, Act

POST-INCIDENT:
- Check principal
- Check team
- Secure area
- Document events

COMMON DISTRACTORS:
- Concealment stops bullets (wrong - only cover does)
- Car doors provide cover (wrong - bullets pass through)
- Run towards police when they arrive (wrong - stay still, hands visible)
- Property is second priority (wrong - it's lowest)
- Run in same direction as vehicle (wrong - run perpendicular)
