WHY THIS CHAPTER MATTERS
------------------------
Venue security encompasses all the measures taken to protect the principal 
at fixed locations - from arrival through their time at the venue to 
departure. Understanding cordons, sterile areas, arrival/departure 
procedures, and event security is essential for professional CP operations.

TOPIC 2.4.1: PRINCIPLES OF VENUE SECURITY
--------------------------------------------------------------------------------

What is Venue Security:
- Protection at fixed locations
- Control of environment
- Managing access
- Enabling principal's activities
- Emergency preparedness

Goals of Venue Security:
- Principal safety
- Controlled environment
- Early threat detection
- Response capability
- Smooth operation

Defence in Depth:
- Multiple security layers
- Each layer adds protection
- Failure of one doesn't compromise all
- Time to respond
- Progressive security

Security Zones:
- Public areas (least controlled)
- Controlled areas (some access control)
- Restricted areas (limited access)
- Sterile areas (maximum control)
- Principal's immediate space

Risk-Based Approach:
- Assess venue threats
- Proportionate measures
- Resource allocation
- Balance security/access
- Client requirements

TOPIC 2.4.2: CORDONS AND PERIMETERS
--------------------------------------------------------------------------------

What is a Cordon:
- Security boundary
- Access control point
- Screening barrier
- Defines security zones

Types of Cordons:

OUTER CORDON:
- Furthest from principal
- First line of security
- Largest area
- Initial screening
- May be public-facing
- Deterrent effect

INNER CORDON:
- Closer to principal
- Higher security level
- Verified access only
- Smaller area
- More thorough checks

IMMEDIATE AREA:
- Principal's location
- Highest security
- CPO controlled
- Very restricted
- Sterile environment

Cordon Functions:

DETERRENCE:
- Visible security
- Discourages approach
- Psychological barrier

DETECTION:
- Identify threats early
- Screen individuals
- Observation point

DELAY:
- Slow any attack
- Create response time
- Physical barriers

RESPONSE:
- Positioned resources
- Communication link
- Quick reaction

Manning Cordons:
- Appropriate staffing
- Communication equipped
- Clear instructions
- Authority to act
- Backup available

Temporary vs Permanent:
- Event cordons (temporary)
- Residential cordons (permanent)
- Scalable approach
- Resource dependent

TOPIC 2.4.3: STERILE AREAS
--------------------------------------------------------------------------------

What is a Sterile Area:
- Searched and cleared
- Access controlled
- No unauthorised persons
- No unchecked items
- Maintained security

Purpose of Sterile Areas:
- Principal can relax security
- Reduced threat level
- Safe environment
- Emergency refuge
- Controlled space

Creating Sterile Areas:

SEARCH:
- Thorough search first
- All areas checked
- IED awareness
- Technical sweep if needed

CLEAR:
- Remove unauthorised items
- Check all spaces
- Under furniture
- Behind fixtures

SECURE:
- Control access points
- Lock unnecessary doors
- Post security
- Monitor continuously

MAINTAIN:
- Nothing enters unchecked
- All persons verified
- Continuous vigilance
- Regular rechecks

Types of Sterile Areas:

HOLDING ROOMS:
- Pre-event waiting
- Privacy for principal
- Briefing space
- Emergency refuge

STAGE AREAS:
- Performance spaces
- Presentation areas
- Controlled backstage

MEETING ROOMS:
- Conference facilities
- Private discussions
- Cleared before use

RESIDENTIAL:
- Principal's home
- Hotel rooms
- Safe rooms

Maintaining Sterility:
- Access control
- Entry logging
- Escort procedures
- No unattended items
- Regular sweeps

If Sterility Compromised:
- Report immediately
- Re-search area
- Assess threat
- Consider relocation
- Brief principal

MUST GET CORRECT:

CORDONS:

STERILE AREA:
- Searched and cleared
- Access controlled

DEFENCE IN DEPTH:
- Multiple security layers
- Each layer adds protection
- Progressive security
- Time to respond

ARRIVAL SEQUENCE:
- Approach → Arrival → Debus → Entry
- SAP provides security

DEPARTURE SEQUENCE:
- Preparation → Movement → Embus → Depart

- Quick transition
- Minimum exposure

- Private waiting area
- Searched before use
- Comfortable, with communications
- Can be emergency refuge

SAFE ROOM/PANIC ROOM:
- Emergency refuge
- Reinforced construction
- Communications available
- Last resort during attack

CORDON FUNCTIONS (4 Ds):
- Deterrence
- Detection
- Delay
- (Response capability)

COMMON DISTRACTORS:
- Sterile means medical (wrong - searched and controlled)
- Inner cordon is furthest (wrong - outer is furthest)
